# Profile Page Project
This project is a web-based profile page designed to showcase personal information, skills, academic activities, courses, volunteer experiences, and useful links. The profile page features a sidebar navigation menu that allows users to switch between different sections, ensuring a seamless and engaging user experience.

# Key Features
Responsive Layout: The page is designed to be responsive and adjusts to different screen sizes.

Sidebar Navigation: A fixed sidebar with navigation buttons to switch between sections (Home, About, Academic Activities, Courses, Links).

Background Image: A consistent background image that covers the entire content area, providing a visually appealing backdrop.

Section Content: Each section contains detailed information about personal skills, academic achievements, volunteer experiences, and more.

Interactive Elements: Navigation buttons with hover effects for improved user interaction.
